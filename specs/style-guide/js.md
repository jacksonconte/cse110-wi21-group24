**Rules:** https://standardjs.com/rules.html

**Exceptions:**

- "curly"
    - Always use curly braces for blocks.
- "semi"
    - Always terminate a statement with a semicolon.

**Using ESLint in VSCode:**

- Install and enable the ESLint extension in VSCode.
- USE the .eslintrc.json configuration file in OUR REPO. Don't use other configs!
